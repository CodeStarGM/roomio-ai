# 🏠 Roomio AI

See your floor plan, beautifully rendered. ✨

Roomio AI lets you upload a simple 2D floor plan (hand-drawn or digital), and instantly transforms it into a stunning, fully furnished 3D visualization — same layout, but brought to life with depth, materials, and style.

**No sign-up required** · Built for architects, designers & dreamers.

## ⚡ Features

- Upload 2D floor plans (hand-drawn or digital)
- AI converts layouts into realistic 3D renders
- Adds furniture, textures, and depth automatically
- Clean before vs after comparison view
- No signup required — just upload and visualize

## 🚀 How to Use

1. Upload your 2D floor plan
2. Let the AI process and generate the render
3. View the side-by-side comparison
4. Download your result before refreshing

## 💡 Use Cases

- Architects visualizing concepts quickly
- Interior designers exploring layouts
- Clients understanding spaces better
- Anyone turning ideas into visual reality

## 🛠 Tech Stack

This app is built with modern frontend technologies and AI integration:

- **React.js** – UI library for building interactive user interfaces
- **Vite** – Fast development and build tool for modern frontend apps
- **React Router v7** – Client-side routing for multi-page experience
- **Tailwind CSS** – Utility-first CSS framework for rapid styling
- **TypeScript** – Typed JavaScript for safer and scalable code
- **AI Model** – Gemini
